    # Process Parameters

    Instruction 说明

    表示在步骤的预览、活动和完成视图中可见的说明文本

    属性 类型 说明
    Column 1 HTML 文本 要显示的说明文本。最大长度为 2000 个字符（包括 HTML 标签）
    Column 2 HTML 文本 未使用
    Column 3 HTML 文本 未使用
    List of options 选项列表

    定义在执行期间可供选择的选项

    属性 类型 说明
    选项 文本（结构化） 将可用选项定义为键/显示文本值对。键和显示文本在一个 phase 内都是唯一的

    为了输入键/显示文本对，系统提供了一个选项列表编辑器

    Expected value configuration 预期值配置

    定义是否必须根据预期值检查执行期间选择的实际选项

    属性 类型 说明
    启用 标志 控制是否执行检查，如果启用，确保已设置了预期值定义中的预期值
    显示 标志 控制预期值是否为执行时显示，预期选项加下划线显示。
    风险评估 选择列表 定义异常的风险级别从而控制相关的签名权限
    异常文本 文本 定义使用的异常描述在异常处理期间和批记录内。最大长度为 250 个字符
    Expected value definition 预期值定义

    如果启用了相应的检查，则将所需选项的键定义为预期值

    属性 类型 说明
    预期值 字符串 定义预期值
    默认值 字符串 定义预选项目
    One-click completion 一键完成

    定义当操作员在执行期间选择一个选项时是否确认步骤并因此自动完成。如果配置了预期值，则只有当所选选项与预期值匹配时，该步骤才会自动完成

    属性 类型 说明
    启用 标志 控制步骤是否自动完成，选择一个选项后自动完成
    Correct value 纠正值

    表示可从导航工具访问的完成后异常。该异常允许操作员更正步骤处于活动状态时选择的选项。它涵盖了操作员由于读取错误而选择了不正确的选项，但在检测到错误之前已经确认并完成了该步骤的事件

    属性 类型 说明
    风险评估 选择列表 定义异常的风险级别从而控制相关的签名权限
    异常文本 文本 定义使用的异常描述

    # Output Variables

    Option key 选项键值

    **数据类型：**String 文本，用于显示一个预定义的字符序列

    **用途：**输出变量提供所选选项的键值

    Option text 选项文本值

    **数据类型：**String 文本，用于显示一个预定义的字符序列

    **用途：**输出变量提供所选选项的显示文本

    Identifier 步骤标识符

    **数据类型：**String 文本，用于显示一个预定义的字符序列

    **用法：**输出变量提供步骤的标识符

    Instance count 步骤执行次数

    **数据类型：**Long 整数

    **用法：**输出变量提供实例数的计数

    Start time 开始时间

    **数据类型：**Timestamp 时间戳，用于显示日期和时间，用于时间相关计算。要在阶段属性中使用时间戳，您必须确保它具有匹配的数据类型，因此要在说明文本中显示它，必须将其转换为字符串

    **用途：**输出变量提供步骤的开始时间

    Completion time 完成时间

    **数据类型：**Timestamp 时间戳，用于显示日期和时间，用于时间相关计算。要在阶段属性中使用时间戳，您必须确保它具有匹配的数据类型，因此要在说明文本中显示它，必须将其转换为字符串

    **用途：**输出变量提供步骤的完成时间

    # Example：计算两个时间点之间的时间段

    // Example:
// Sample Phase with Start time = 14-Nov-2014@10:15
// Sample Phase with Completion time = 14-Nov-2014@11:47
// The duration is to be displayed in minutes.
convertToDisplayString(
  convertTo(
    {Sample Phase}.{Completion time}-{Sample Phase}.{Start time},
  "min"
  )
)
+ " min"
// As result of the expression, the system displays "92 min".
