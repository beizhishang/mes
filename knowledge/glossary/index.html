    # IN

    # Innovent

    • 信达生物的缩写，用在标识phase的使用范围，如D Weight(IN)表示为信达开发的分发称量phase

    # RA

    # Rockwell Automation

    • 罗克韦尔，MES的供应商

    • Rockwell是一家提供工业自动化和信息解决方案的全球科技公司，服务范围涵盖了汽车、食品和饮料、生命科学以及石油和天然气等多个行业。该公司成立于1903年，总部位于美国威斯康星州的密尔沃基市。

    • Rockwell的产品和服务包括控制系统、软件、传感器和其他工业自动化组件，帮助制造和工业运营提高生产效率、效率和安全性。该公司的软件解决方案旨在帮助客户优化其生产过程，减少停机时间，并提高产品质量和合规性。

    • Rockwell还提供一系列服务，包括咨询、工程和培训，以帮助客户充分利用其自动化投资。该公司在全球拥有业务和客户，覆盖了100多个国家。

    • 近年来，Rockwell一直致力于扩大其软件和数字解决方案的产品组合，帮助客户利用数据和分析的力量来改善其运营。该公司还一直在投资新兴技术，如人工智能和物联网（IoT），以帮助客户在日益数字化和互联的世界中保持领先地位。

    # SCL

    # Solution China Library

    • 罗克韦尔的中国解决方案库

    # UDA

    # User Definition Attribute

    • 用户自定义的属性，在软件开发时，预留了一些未使用的字段，这些字段可以定制成客户想要的对象属性

    # BB

    # Building Block

    • 构建块在主配方或主工作流的任何级别上的单个结构组件，可用于构建配方或工作流结构：过程、单元过程、操作或阶段。

    # AS

    # Activity Set

    • 活动集，生产执行的功能页面

    # AE

    # Administration Extention

    • 管理员扩展，生产管理界面的功能页面

    # PMC

    # Production Management Client

    • Rockwell PharmaSuite Production Management Client是一款软件解决方案，旨在帮助制药和生物技术公司管理和优化其生产过程。该软件提供了一套全面的工具和功能，使用户能够在实时监控下管理批量生产并跟踪产品质量和生产绩效。

    • 使用PharmaSuite Production Management Client，用户可以轻松地管理和监控批量生产活动，包括物料跟踪、排程和执行。该软件还包括高级功能，用于管理质量控制和合规性，包括电子批记录和实时监测关键过程参数。

    • PharmaSuite Production Management Client与PharmaSuite软件套件中的其他模块（如过程设计和配方管理）集成，为生产管理提供完整的端到端解决方案。该软件还支持与其他制造系统（如SCADA和MES）集成，以提供无缝统一的生产环境。

    • PharmaSuite Production Management Client包括一系列先进的分析和报告功能，使用户能够实时分析生产数据，以识别趋势、优化流程并改善整体生产绩效。

    # PEC

    # Production Execution Client

    • Rockwell PharmaSuite Production Execution Client是一款软件解决方案，旨在帮助制药和生物技术公司管理和优化其生产过程。该软件提供了一套全面的工具和功能，使用户能够在生产生命周期中管理批量生产并跟踪产品质量。

    • 使用PharmaSuite Production Execution Client，用户可以轻松地管理和监控批量生产活动，包括物料跟踪、排程和执行。该软件还包括高级功能，用于管理质量控制和合规性，包括电子批记录和实时监测关键过程参数。

    • PharmaSuite Production Execution Client与PharmaSuite软件套件中的其他模块（如过程设计和配方管理）集成，为生产管理提供完整的端到端解决方案。该软件还支持与其他制造系统（如SCADA和MES）集成，以提供无缝统一的生产环境。

    • 总的来说，PharmaSuite Production Execution Client有助于制药和生物技术公司提高生产效率，降低成本，并确保产品质量和合规性。

    # PD

    # Process Design

    • Rockwell PharmaSuite Process Design是一款专门为制药和生物技术行业设计的软件解决方案，旨在帮助简化和优化新产品的设计和开发过程。该软件提供了一套全面的工具和功能，使用户能够从最初的概念到最终生产创建、编辑和管理复杂的过程设计。

    • 使用PharmaSuite Process Design，用户可以创建具有详细过程流程、设备和材料要求的过程模型。该软件还包括高级模拟和分析工具，使用户可以在实施之前测试和优化过程设计，有助于降低错误风险并提高效率。

    • PharmaSuite Process Design还与PharmaSuite软件套件中的其他模块（如批处理管理和配方管理）集成，为过程设计和开发提供完整的端到端解决方案。总的来说，PharmaSuite Process Design有助于加速制药和生物技术公司的产品开发，降低成本，提高质量和合规性。

    # ERP BOM

    # Enterprise Resource Planning Bill of Materials

    • ERP BOM，即企业资源计划物料清单，是制造产品所需的所有物品、组件和材料的全面清单。它包括每个物品的描述和计划数量等相关属性。在 PharmaSuite 中，当您根据 ERP BOM 编辑主配方时，系统会在单独的节点上提供所有 ERP BOM 物品及其相关属性，这有助于管理和组织生产过程所需的必要组件和材料。
