    Question & Answer

    以下是我们收集的一些常见的问题，如果您在此处没有找到您想要的答案，请尝试在文档中寻找相关内容。如果您的问题仍然无法解决，请单击右下角的对话按钮与我们联系。我们将尽快为您提供答复。

    # 没有种子扩增的工作站

    操作业务流程培训类

    根据 PIQ 脚本中的工作站分配清单登录，最新的分配清单已经发在群里，后期 SOP 同步更新附录。

    # 清洗了 6 个转接头，但是 6 个装在一包里，应该打几个已清洁标签？

    操作业务流程培训类

    现在线下一包只有一个已清洁标签，所以在 MES 里也只需要打一个；装配的时候扫描标签是记录清洁的批号以及判断 CHT 时间，和标签对应的器具数量没有关系，所以只要用这一包的接头就重复扫这个已清洁标签即可。

    # 标签重新打印怎么操作？

    操作业务流程培训类

    生产管理界面-报表管理-根据条件筛选出标签，签名备注重新打印。

    # 最终淋洗间工作流没结束，清洗间不能再开下一个工作流？

    处方类

    目前工作流的设计是工作流结束才解绑清洗间；工作流下次优化时在清洗间操作结束的最后一步就把清洗间解锁掉。

    # 清洗间有 3 台平板，3 个打印机，该怎么对应使用？

    IT 网络硬件

    其中 2 台平板用来执行清洗或者灭菌的工作流，分别将默认打印机设置为装绿色标签的两台；另外 1 台平板用来执行组装的工作流，默认打印机设置为黄色标签的一台；因为这一台平板在选择打印标签时会频繁切换打印机，如果末次切换为黄色打印机，下一次再拿来走清洗的工作流，就会导致清洗标签通过黄色标签纸打印出来。

    # MES 中执行秤校对工作流、消毒剂配制工作流不会打印出过程中标签，现在线下业务是需要打印标签的。

    操作业务流程培训类

    秤校对和消毒剂配制工作流一样，是系统有自动判断是否成功的逻辑的，所以不会通过打印标签来证明； 线下记录要备注，解释 MES 的逻辑，然后说一下已经判定是否成功，FS 的逻辑已经经过 PQ 验证了，且该备注需要双签。 消毒剂配制可以把 MES 的工作流打印出来，作为这份配置记录的附件。

    # 器具清洗批号发生跳号问题

    系统问题

    到现场确认发生跳号问题，等待 RA 进行 BUG 修复，临时将跳过的批号在 logbook 里备注一下，备注好跳过的批号和跳过的原因；BUG 修复后在验证环境验证完成后到现场配合生产进行测试，确认功能正常。

    # 物料接收时，扫描物料需要反映几分钟才能出来。

    IT 网络硬件

    与业务多次去车间进行确认，发生情景无规律，现场已经测试过扫码枪、手输物料编号都存在卡顿现象，基本可以排除扫码枪与终端蓝牙问题。需要 IT 与 RA 再综合找寻问题原因。

    # 器具待清洁标签打印出来器具名称显示是 NA，正常吗？

    操作业务流程培训类

    正常，如果器具有多个种类，打出来的标签上器具名称会显示 NA；如果一个种类，则会显示器具名称。

    操作业务流程培训类

    没确认签名，这张标签在系统里是不存在的，器具也没法接收。需要：1.该标签作废；2.重新做一遍器具回收，打印待清洁标签-签名确认。

    # 打印出的待清洁标签上，物品种类需要根据实际情况勾选吗？还是不用了

    # 操作过程，平板异常关机，重启后进入生产执行，对话框显示请联系维护人员解除锁定，是要找主管解锁吗？

    IT 网络硬件

    找主管或者经理解锁，解除锁定要确认下解锁的信息，与图中显示的设备编号相关的，不要解锁到其他人的；解除锁定前先发个图片在群里确认解除锁定的对象对不对。

    # BOM 外物料无法在 MES 中做物料接收

    操作业务流程培训类

    临时解决方案：如果是 911 领的没有写收货库位，需要通过 MB22 修改后保存，将领料单号发在群里给大家重新发货。 根本解决方案：BOM 外的物料原来不使用 MES 时，一般都不选择库位，现在用 MES 需要选择 SAP 对应的库位，不然 MES 接收不了。

    # 打印机无法成功打印标签，内容偏右，字体不清晰等

    IT 网络硬件

    • 提供打印机参数调整教程，长按桌面打印机，调整打印机设置项，按照教程进行设置; 调整单位、调整标签尺寸、调整打印速度、浓度、偏移量；
    • 设置之后如果还是无法打出标签，提供 Surface 编号，IT 远程解决。

    # 平板电源键一按就会关机，员工经常误操作导致账号被锁定，能否帮忙设置取消该设定，改成屏幕息屏。

    需要域控推攻略

    # 清洗结束了，淋洗间这边工作流没有下一个操作层级的显示。

    操作业务流程培训类

    这个工作流是分为两个操作层级的，清洗间的操作结束之后，淋洗间的操作才会显示；现在清洗间的操作还没有执行完，还没有上传清洗报告、记录电导率和温度，记录结束时间；所以工作流无法到淋洗间的操作层级。

    # 无法扫入房间/扫描显示乱码

    操作业务流程培训类

    几乎所有扫描二维码无法正确扫描的原因都是输入法的问题，用扫码枪时需确认为英文输入法，并且切换为小写。

    # 培养基配制间无法打印出待清洁器具的标签。

    操作业务流程培训类

    IT 人员现场支持调整，网线插错网口。

    # 淋洗间已清洁器具标签打印不清晰。

    IT 网络硬件

    发送打印机参数调整教程并视频支持，成功打出清晰的标签。

    # 中试研发的工单不小心误点了，如何取消？

    操作业务流程培训类

    中试研发工单误点不可以取消，因为还没有执行完，只要把工单挂在任务栏就行，需要执行的时候再点进去。

    # 称量标签的大小为什么变了？标签存在大量空白区域这样设计合理吗？配液的纸质批记录没有足够的空间张贴怎么办？提前不给配液通知，标签怎么处理？

    操作业务流程培训类

    MES 称量物料的标签信息比较多，下面可能还需要放 GHS 的标识，标签下方预留了 GHS 标识的空间，为了所有的信息都能正常显示，所以称量标签当时与称量同事一起定下了 100*120mm 的尺寸； 贴在白纸上附件附在批记录中，或者像贴便签纸一样贴在批记录上，QA 同事做了详细解答。

    # 器具装载选错了，等到填清洗装载数量的时候发现没有想要的器具，回头修改选择的装载可以吗？

    操作业务流程培训类

    回头修改选择的装载只能修改选择记录的装载值，但是当前已经显示的清洗装载条目是根据第一次的选择带过来的，是不会根据修改值而改变的； 解决方案 1：终止工作流并记录异常，重新开一个工作流进行清洗，但是这样会导致 1 个清洗批号被系统占用； 解决方案 2：清洗选择装载的器具，把这个工作流给正常执行完。

    # 器具扫描没有二维码，如何进行下一步？

    操作业务流程培训类

    针对这一步，签名终止 phase，相当于跳过该步骤的动作；所有终止动作请于主管经理、MES SME 确认。

    # 器具清洗间的平板被切换到称量的工作站，但是器具人员有没有切换工作站的权限，也没有登录称量工作站的权限。

    操作业务流程培训类

    临时措施：使用 MES 系统管理账号切换工作站到器具，然后再让器具人员登录账号； 根本措施：以后大家第一次按照系统分配登录工作站之后，不要再切换了，也不要去切换其他区域终端的工作站，不然就很有可能会被锁定或者导致一些操作没法执行。

    # 非 BOM 物料消耗时，1 包里面有 10 个，填消耗数量 1，结果 1 包都消耗完了。

    操作业务流程培训类

    物料的单位是 SAP 传过来的，非 BOM 物料填消耗数量时对应的单位就是传过来的单位，MES 无法进行修改，所以填消耗数量时需要人工换算。例如 1 包 10 个，如果这次用了 1 个，那就填入 0.1bag。

    # 溶液有效期在标签上显示不正确。

    处方类

    临时措施：工单中进行异常备注，备注正确的溶液效期；溶液的标你们暂时需要在标签上手改一下效期； 根本措施：溶液的效期不在一期的设计管理范围内，所以处方中未对相关参数进行调整，要显示正确的效期需要修改处方相关参数并升版处方；具体讨论会已于 2022.11.25 与大家沟通完成，PPQ 结束前按照沟通结果升版处方。

    # 物料没有做立即转移，连廊这个库存位置有很多物料

    操作业务流程培训类

    做物料接收时，如果物料即刻拿到使用区域，那就要做立即转移，将物料在 MES 系统中从连廊的库位转移到实际生产区域的库位。

    # 中试研发称量物料如何发料？

    操作业务流程培训类

    中试研发称出来的物料不需要在 MES 系统中做物料的交接发料收料操作，因为 MES 系统不会产生对应的物料库存。

    # 工作流执行中，平板断电异常闪退，会话解锁后，工作流还是无法执行

    操作业务流程培训类

    临时措施：重新开启一个房间清洁工作流，线下也重新进行清洁记录；具体操作截图并进行详细注释说明。 根本措施：因为执行到一半没有电量闪退了，导致后台数据出错，后面大家在使用平板的过程中，需要注意下电量是否充足，不然可能还会有同样的问题发生。

    没差，这个时间是使用结束时间，不是回收时间，看错台账条目了。

    # 中试研发称量物料上的标签缺少批号和效期信息？商业化生产的称量物料标签是否也缺少这些信息？

    系统问题

    称量的标签有 2 种，商业化生产的物料称量标签上所有信息都齐全；中试研发的标签，在 MES 系统中我们只扫物料进行扣账，没有产出，所以溶液批号和效期信息系统无法生成或抓取，效期信息原来的手写标签也没有； 措施：已提交 MES 系统变更申请单 MESCR-2022-0001，在标签上增加批号与复核人的表头，信息由人员手写。

    # 中试研发物料扣账如何操作？

    操作业务流程培训类

    中试研发扣料操作如下：1.根据原液批号搜出中式研发的工单；2.点开任务栏里的中式研发步骤；3.扫码或手输物料编码查询出需要扣账的 BOM 物料；4.确认后到下一步，扫描 BOM 物料的标签二维码并进行消耗；5.整个 306 产品里的 BOM 物料扣账全部扣完就点结束，否则需要点循环，工单可以一直挂在任务栏，直到所有 BOM 物料扣账完成； 具体操作已经截图并做操作详细说明。

    # 工艺袋未确认，应如何做？工艺袋不在此处使用麻烦考虑我们物品使用逻辑情况；BOM 内称量物料忘记投料却点了停止怎么办？

    操作业务流程培训类

    # 清洗间上传清洗报告错了，到淋洗间了才发现，如何替换？

    操作业务流程培训类

    由于淋洗间的工作站无法访问清洗报告的文件夹，所以只能先解绑淋洗间的操作，然后在清洗间登录淋洗间的工作站，然后把清洗报告重新上传；具体操作已详细说明并指导完成。

    # 消毒剂配制，打印出来的标签是空白内容？

    操作业务流程培训类

    在系统的报表管理-标签重打印中查询标签是否已经在系统中生成，已经生成说明是打印机的问题，需要 IT 同事协助处理； 打印标签前，需要确认打印机是开机状态，不然等到打印那一步再开启打印机可能就会导致标签无法正常打出。

    # 称量后物料效期规定是 ≤30 天，MES 标签的时间计算是否应为+29 ，目前标签上是+30 的

    操作业务流程培训类

    由于系统是按照小时精准管理，所以物料效期维护的是 720 小时；经过开会讨论，将效期时长缩短为 29*24h=696h，PPQ 结束前统一修改；

    # 标签重新打印功能，如果第二天打印今天的标签，配制日期会是今天还是明天？

    操作业务流程培训类

    会是的，系统中已经产生了标签，重新打印的标签和原标签是一模一样的，不会改变标签内容，只会增加重打印标识和时间。

    # 如何查询已经提交过的退料单信息？

    操作业务流程培训类

    要查所有的退料单历史记录在生产管理这里查。物料退料的状态有三种（进行中，已取消，已完成）；退料需要在生产执行执行 2 步：1.在【物料退料】功能里执行完操作，退料的状态会是“进行中”；2.需要在【退料单查询】功能里打印出退料单，这样才算完成退料的步骤，退料的状态会变成“已完成”，在这个功能里面只会显示“进行中”的退料明细；注：如果误操作需要取消退料，在【退料单查询】里面取消之后，退料的状态会变成“已取消”。

    # 灭菌工作流中“确认物品摆放点位，名称，最大装载与 SOP 点位图是否一致”的描述，与现场实际操作不符
